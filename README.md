# 🤖 AIHelpMath

¡Bienvenido a **AIHelpMath**! 

## 📖 Contexto de la Aplicación

**AIHelpMath** es una plataforma web diseñada para asistir a los usuarios en el aprendizaje y la resolución de problemas matemáticos mediante el uso de Inteligencia Artificial. Esta plantilla proporciona la estructura base del frontend para el proyecto.

El sistema cuenta con un flujo completo de usuario que incluye páginas de inicio (Landing Page), registro, inicio de sesión y un panel de control (Dashboard) principal donde los usuarios interactúan con la IA para obtener explicaciones paso a paso de sus dudas matemáticas.

### 🚀 Características Principales
* **Asistente de IA Matemático:** Integración con modelos de lenguaje para resolver problemas y explicar conceptos.
* **Interfaz Moderna e Intuitiva:** Flujo de navegación desde el registro hasta el dashboard.
* **Diseño Responsivo:** Adaptable a diferentes dispositivos para estudiar desde cualquier lugar.

### 📂 Estructura del Proyecto
* `index.html`: Landing page y presentación del proyecto.
* `login.html` / `registro.html`: Vistas de autenticación de usuarios.
* `dashboard.html`: Panel principal donde el usuario interactúa con la plataforma.
* `/css/` y `/js/`: Estilos y scripts correspondientes al frontend.
* `/assets/`: Imágenes y recursos gráficos.
